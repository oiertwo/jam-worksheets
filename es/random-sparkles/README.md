## Preparación

- Si usas Sense HAT, conectalo a la Pi antes de encenderlo.
- Si usas el emulador de Trinket, ejecuta `sense.clear()` antes de empezar:
    ```python
    from sense_hat import SenseHat
    
    sense = SenseHat()
    
    sense.clear()
    ```

    despues borra el codigo y empieza desde cero.
